# StayGuide Pro - Demo Guide

## Quick Start Demo

### 1. Start the Server
```bash
cd stayguide-prototype
npm run dev
# Running on http://localhost:3001
```

### 2. Login to Dashboard
- Go to: http://localhost:3001/auth/login
- Use ANY email/password (demo mode)
- Example: demo@stayguide.pro / demo123

### 3. Explore Features

#### Dashboard Overview
- See property performance metrics
- Email capture analytics
- Revenue tracking
- Conversion rates

#### Create New Property
1. Click "Add Property" button
2. Fill in basic info
3. Click "Generate with AI" for auto-fill
4. Save & continue to guide editor

#### Guide Editor (Coming Next)
- Drag & drop blocks
- WiFi with email capture
- AI content generation
- Mobile preview

### 4. Guest Experience
- Visit: http://localhost:3001/g/lakeside-paradise
- See the enhanced PWA guide
- Test email capture via WiFi
- Try the AI concierge chat
